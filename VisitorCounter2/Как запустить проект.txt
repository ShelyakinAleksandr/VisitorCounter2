необходимо:
1) Visual Studio 2019 Версии 16.9.1 (дальше VS)
3) .Net Core 3.1
2) Субд Mysql Версии 8.0.27

Запуск
1) Раскатить дамп базы из папки в проекте DataBase (описание как раскатить там же)
2) Открыть файл VisitorCounter2.sln через VS
3) Изменить подключение к базе в файле appsettings.json, DefaultConnection
4) Запустить проект через IIS Express
5) Откроется браузер на странице http://localhost:13706 (номер порта(:13706) может отличатся)
6) Добавить к текущему адресу страницы /swagger
7) Откроется страница http://localhost:13706/swagger/index.html (на этой странице должны быть отображены Контролеры проекта)
